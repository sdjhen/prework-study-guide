HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Meta tags are useful for Search Engine Optimisation (SEO) and help websites appear in searches on Google and other similar search engines.

CSS

  • Margin indicates how much space we want around the outside of an element.
  • Padding indicates how much space we want around the content inside an element.
  • An element can be centered by applying the block property to it and and setting the margin to auto. Make sure the width is under 100%!

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it
  • It is good practice not to code to the main branch straight away - using branches prevents bugs from entering the main codebase and disrupting the project!
